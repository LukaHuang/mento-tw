演講大綱

在當今快速變化的技術世界中，Docker 已成為不可或缺的工具，尤其對於 Web 工程師來說，更是如此。為什麼 Web 工程師需要學習 Docker？透過這場線上小聚，你將了解 Docker 不僅能夠提升個人開發效率，同時也促進前端和後端工程師之間的合作。

  • Docker 如何使整個開發流程更加簡化並提升效率？
  • 容器技術如何幫助前端和後端工程師更好地合作
  • 前端工程師是否能單獨使用 Docker image 來發布專案？
  • 使用 Docker 時應注意哪些重要事項？

透過小賴帶來的這場 Docker 演講，無論你是前端還是後端工程師，都能學到容器化的核心技能，使你能在工作中處於領先地位。立即報名參加，開啟你的 Docker 學習之旅！

講者介紹

賴怡玲 (小賴) 是一位熱愛網站開發的一條龍工程師，擅長從兵荒馬亂、資源有限的團隊中做出些什麼來。特別喜歡後端開發、容器技術跟 AWS 雲端服務。同時她也是 2022 年 iThome 鐵人賽 DevOps 組冠軍，著有《Docker 實戰 6 堂課：56 個實驗動手做，掌握 Linux 容器核心技術》，目前擔任教育訓練的工作，推坑更多人加入 Web 開發與 SRE 領域。

演講與教學經驗

  • 2020 Laravel Conf 講者《 我們與Serverless 之間的距離 》
  • 2023 Web Conf 講者 《 Docker Workshop 》
  • 2023 Modern Web Conf 講者《 Docker Workshop 》
  • 曾在 AppWorks School 擔任資深技術導師，負責後端班與 SRE 班
  • 曾在資展國際兼職擔任 NodeJS 課程講師

預期收穫

  • 學會怎麼用後端工程師提供的 Docker image 來啟動後端環境。
  • 串接後端 API 的時候，用 Docker container 管理環境會更有效率。
  • 能夠把自己的開發環境 Build 成 Docker image 。
